📋 DEPLOYMENT STEPS (5 minutes)
────────────────────────────────────────────────────────────────

1. PUSH TO GITHUB:
   $ git commit -m "Ready for deployment"
   $ git push

2. DEPLOY ON VERCEL:
   → Go to: vercel.com/new
   → Click "Import Git Repository"
   → Select: everypasswordsucks
   → Click "Deploy"
   → ✅ Live in 2 minutes!

3. ADD CUSTOM DOMAIN (Optional):
   → Vercel Dashboard → Settings → Domains
   → Add: everypasswordsucks.com
   → Follow DNS instructions
   → ✅ Live at your domain in 1-24 hours

═══════════════════════════════════════════════════════════════

📊 TRAFFIC TRACKING SETUP (3 minutes)
────────────────────────────────────────────────────────────────

GOOGLE ANALYTICS (Primary - Most Important):

1. CREATE ACCOUNT:
   → analytics.google.com
   → Create property
   → Copy Measurement ID: G-XXXXXXXXXX

2. ADD TO VERCEL:
   → Vercel → Settings → Environment Variables
   → Add: NEXT_PUBLIC_GA_MEASUREMENT_ID = G-XXXXXXXXXX
   → Redeploy

3. VERIFY WORKING:
   → Visit your live site
   → Google Analytics → Realtime
   → See yourself as visitor!

🔧 TROUBLESHOOTING
────────────────────────────────────────────────────────────────

SITE NOT LOADING:
→ Check Vercel deployment logs
→ Try: npm run build (locally first)

ANALYTICS NOT SHOWING DATA:
→ Wait 24 hours for first data
→ Check "Realtime" view (shows immediately)
→ Disable ad blockers
→ Verify environment variable in Vercel

DOMAIN NOT WORKING:
→ Check DNS: dnschecker.org
→ Wait 24-48 hours
→ Use Vercel nameservers (recommended)
